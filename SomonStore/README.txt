Peshsaf - учебный проект интернет-магазина

Что есть в проекте:
- регистрация и вход через Session
- роли Admin / Customer
- каталог товаров
- поиск и фильтр по категориям
- корзина пользователя
- оформление заказа
- админ панель для товаров: добавить / изменить / удалить

Тестовые аккаунты:
admin / 123
user / 123

Обычно достаточно:
1) удалить старую базу PeshsafDb в PostgreSQL
2) запустить проект заново
