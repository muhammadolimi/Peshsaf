SomonStore - учебный проект интернет-магазина

Что есть в проекте:
- регистрация и вход через Session
- роли Admin / Customer
- каталог товаров
- поиск и фильтр по категориям
- корзина пользователя
- оформление заказа
- админ панель для товаров: добавить / изменить / удалить
- DataSeeder с тестовыми данными

Тестовые аккаунты:
admin / 123
user / 123

Важно:
Если у тебя уже была старая база данных от прошлой версии проекта, удали её один раз, потому что в новой версии добавлено поле ImageUrl у товара и изменена логика сидера.

Обычно достаточно:
1) удалить старую базу SomonStoreDb в PostgreSQL
2) запустить проект заново

В этом проекте используется EnsureCreated(), поэтому на чистой базе таблицы создадутся автоматически.
